# Stand By Schedule Web

React + Vite frontend for the stand-by schedule system.

## Quick Start
```bash
bun install
bun run dev
```

## Environment
Copy `.env.example` to `.env` and fill in:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `VITE_API_URL` (points to the API repo)

## Related Repo
The backend API lives in `C:\typescript\stand-by-schedule-api`.
